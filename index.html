How to use this guide

This guide is split into short sections, listed in the menu on the left. You do not need to read it front to back. When you want to do something, find the matching section and follow the numbered steps. Look for the coloured boxes as you go:

Do

A recommended action or a safe thing to change on your own.

Good to know

Helpful background or a shortcut that makes a task easier.

Take care

Something that works, but where a small mistake can cause a visible problem. Read before changing.

Do not

Something that can break part of the store. Leave these to your developer or ask first.

Five golden rules

If you remember nothing else, remember these.

1. Always Save

Changes are not live until you press the Save button (usually top right). If you leave a screen without saving, your edits are lost.

2. Active means visible

A product set to Active can be seen and bought by customers. Draft hides it. Use Draft while you are still working on something.

3. When unsure, duplicate first

Before experimenting on a real product, use Duplicate to make a copy, set the copy to Draft, and practise on that. Delete the copy when done.

4. Configurator products are shared

The option menus (metaobjects) are reused across many products. Editing one choice can change several products at once. The Configurator section explains this before you touch anything.

5. Do not edit theme code or delete metaobjects

The store's design and the configurator engine rely on code and data set up for you. Editing Online Store › Themes › Edit code or deleting metaobjects can break pages. Ask your developer for anything in those areas.
