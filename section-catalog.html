Reference

Section catalog

Every building block available in your theme, grouped by where it is used. This is a reference: skim it to see what exists, then use the page-by-page guides to edit each one.

Your theme is built from 72 sections. Most pages are assembled by stacking these sections in the theme editor. The "Settings" column shows how many options each section has, and "Appears on" shows where it is currently placed.
